Message

世界経済の発展には必要不可欠な物流。その中でも海運は、低コスト・大量輸送の筆頭として人々の生活に深く根付いてまいりました。 私ども平安海商は“安全・確実”な海運を通じてより高度なサービスをお客様に提供することを基本理念として日々の企業活動に鋭意努力しております。海運を通じて世界の豊かさに貢献できるという喜びの反面、その社会的責任を深く感じる次第です。 今後も当社は、数多の経験を糧とした配船能力を強みとしてお客様・社会から信頼される企業であり続けれるよう努めてまいります。 「期待に応え続ける会社」を目指しますので今後も皆様のご愛顧を賜りますとともに、ご指導、ご鞭撻の程宜しくお願い申し上げます。
